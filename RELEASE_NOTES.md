## Que es Maxim Broadcast?

Maxim Broadcast es una plataforma avanzada de produccion audiovisual que transforma OBS Studio en una solucion completa de broadcast profesional. Combina produccion en vivo, edicion de video, inteligencia artificial y automatizacion inteligente en un solo software — con compatibilidad absoluta con OBS Studio y todos sus plugins.

**Ideal para:**
- Estaciones de TV y radio
- Productoras audiovisuales
- Streamers profesionales
- Iglesias y eventos en vivo
- Canales de YouTube y Twitch
- Empresas con comunicacion interna en video
- Educacion a distancia y webinars

## Caracteristicas Principales

### Lower Thirds Profesionales
- Sistema completo de graficos dinamicos para identificacion de personas y segmentos
- Animaciones fluidas y personalizables con timeline de keyframes
- Editor visual integrado con preview en tiempo real
- Templates prediseñados para noticias, deportes, entretenimiento, entrevistas y corporativo
- Soporte para fuentes tipograficas personalizadas y logos

### Sistema de Templates
- Biblioteca de mas de 50 plantillas profesionales prediseñadas
- Motor de plantillas personalizable con variables dinamicas
- Gestor de assets graficos con categorias y etiquetas
- Importacion/exportacion para compartir entre equipos
- Preview en tiempo real antes de aplicar

### Editor de Video Dual
- Timeline no-lineal completo integrado en la plataforma
- **Edicion simultanea mientras se transmite en vivo** — sin interrupciones
- Recorte, empalme y ajustes en tiempo real
- Efectos de video aplicables tanto a contenido pregrabado como en vivo
- Exportacion multi-formato (MP4, MOV, MKV, WebM) sin detener el streaming
- Preview multipista con biblioteca de medios centralizada
- Soporte para proxies y edicion de archivos de alta resolucion

### Produccion Profesional
- Dashboard avanzado con layout preview/program tipo broadcast
- Multiples salidas simultaneas: RTMP, SRT, grabacion local, NDI
- Transiciones profesionales con motor de efectos personalizables
- Mezclador de audio multicanal con EQ parametrico y compresor por canal
- Sistema de escenas avanzado con agrupacion y jerarquias
- Monitoreo de señal con vectorscopio, forma de onda y histograma

### Multi-Camara Avanzado
- Soporte hasta **12 fuentes de video simultaneas**
- Instant replay con control de velocidad variable (0.1x — 4x)
- Auto-switching inteligente basado en analisis de contenido
- Picture-in-picture avanzado con posicionamiento libre y bordes
- Split-screen dinamico con layouts predefinidos y personalizados
- Preset manager para guardar y cargar configuraciones rapidamente

### Inteligencia Artificial
- **Generacion automatica de graficos** usando modelos de IA
- **Subtitulos automaticos en tiempo real** con Whisper (speech-to-text)
- **Auto-framing** con deteccion facial avanzada — sigue al presentador
- **Chroma key mejorado** con IA para deteccion de bordes precisa
- Deteccion de emociones en rostros para analytics
- Generacion automatica de thumbnails optimizados para cada plataforma
- Sugerencias de contenido contextual basadas en el tema

### Automatizacion Inteligente
- **Auto-director con IA** que selecciona camaras automaticamente segun la accion
- Deteccion automatica de silencio para edicion rapida
- **Generacion de highlights** post-transmision con los mejores momentos
- Transcripcion en tiempo real en mas de 10 idiomas
- Analytics de audiencia con metricas avanzadas y graficas
- Ajuste automatico de niveles de audio para mantener consistencia
- Optimizacion de calidad adaptativa segun ancho de banda

- Importacion de perfiles y escenas de OBS existentes
- Mismos formatos de codificacion y protocolos de streaming
- API compatible con integraciones OBS existentes

## Requisitos del Sistema

### Minimos (Desktop)
- **CPU**: Intel Core i5 / AMD Ryzen 5 (4 cores)
- **RAM**: 8 GB
- **GPU**: Compatible con H.264 (Intel HD 530+, NVIDIA GTX 950+, AMD RX 460+)
- **Almacenamiento**: 2 GB libres
- **Internet**: 5 Mbps para streaming

### Recomendados (Desktop)
- **CPU**: Intel Core i7 / AMD Ryzen 7 (8+ cores)
- **RAM**: 16 GB o mas
- **GPU**: NVIDIA GTX 1060+ / AMD RX 580+ con NVENC/VCE
- **Almacenamiento**: SSD con 10 GB libres
- **Internet**: 25+ Mbps para streaming de alta calidad

### Movil (Android/iOS)
- **Procesador**: Snapdragon 855+ / Apple A13+ o equivalente
- **RAM**: 4 GB minimo
- **Almacenamiento**: 500 MB libres
- **Camara**: Soporte para 1080p minimo

## Instalacion

### Windows
```
1. Descargar maxim-broadcast-1.0.0-windows-x64-setup.exe
2. Ejecutar el instalador como administrador
3. Seguir las instrucciones del asistente
4. Abrir Maxim Broadcast desde el menu Inicio o escritorio
```

### macOS
```
1. Descargar maxim-broadcast-1.0.0-macos-universal.dmg
2. Abrir el DMG y arrastrar a Aplicaciones
3. En la primera ejecucion: clic derecho > Abrir (para aprobar Gatekeeper)
4. Conceder permisos de camara, microfono y captura de pantalla
```

### Linux
```bash
# Ubuntu/Debian
sudo dpkg -i maxim-broadcast-1.0.0-linux-amd64.deb
sudo apt-get install -f

# Fedora/RHEL
sudo rpm -i maxim-broadcast-1.0.0-linux-x86_64.rpm

# AppImage (cualquier distro)
chmod +x maxim-broadcast-1.0.0-linux-x86_64.AppImage
./maxim-broadcast-1.0.0-linux-x86_64.AppImage
```

### Android
```
1. Descargar maxim-broadcast-1.0.0-android-universal.apk
2. Permitir instalacion desde fuentes desconocidas en Configuracion
3. Abrir el APK e instalar
4. Conceder permisos de camara, microfono y almacenamiento
```

### iOS
```
1. Disponible proximamente en App Store
2. Para beta testing: instalar via TestFlight o sideload del IPA
```

### Desde Codigo Fuente
```bash
git clone https://github.com/luisitoys12/maxim-broadcast.git
cd maxim-broadcast
npm install
./build.sh
```

## Problemas Conocidos

- Renderizado 4K requiere GPU dedicada con 4GB+ VRAM
- Auto-director IA optimizado para escenas con 2-6 camaras
- Transcripcion multiidioma soporta 10 idiomas principales (español, ingles, portugues, frances, aleman, italiano, japones, coreano, chino, ruso)
- Algunas transiciones complejas pueden causar lag en GPUs integradas
- Android: requiere permisos de camara, microfono y almacenamiento
- iOS: version IPA solo disponible via sideload/TestFlight por ahora
